static

Generates a stream of consecutive integer values, published at regular intervals. When the pipeline is in replay mode, the timing of the messages complies with the speed of the pipeline.

Parameters
pipelineThe pipeline that will run this generator.
startThe starting value.
countThe count of values to generate. Use int.MaxValue if the generator should never stop.
intervalThe desired time interval between consecutive messages. Defaults ot 1 tick.
alignDateTimeIf non-null, this parameter specifies a time to align the generator messages with. If the paramater is non-null, the messages will have originating times that align with the specified time.
Returns
A stream of consecutive integers
