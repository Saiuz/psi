Creates an input receiver associated with the specified component object, connected to an async message processing function. The expected signature of the message processing delegate is:

async void Receive(T message, Envelope env);

Template Parameters
TThe type of messages accepted by this receiver
Parameters
ownerThe component that owns the receiver. This is usually the state object that the receiver operates on. The receivers associated with the same owner are never executed concurrently.
actionThe action to execute when a message is delivered to this receiver.
nameThe debug name of the receiver
autoCloneIf true, the receiver will clone the message before passing it to the action, which is then responsible for recycling it as needed (using receiver.Recycle)
Returns
A new receiver
