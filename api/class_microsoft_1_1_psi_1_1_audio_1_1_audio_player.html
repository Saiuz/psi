Detailed Description

Component that implements an audio player component which plays back a stream of audio to an output device such as the speakers.

This output component renders an audio input stream of type AudioBuffer to the default or other specified audio output device for playback. The audio device on which to playback the output may be specified by name via the AudioConfiguration.DeviceName configuration parameter.

This output component renders an audio input stream of type AudioBuffer to the default or other specified audio output device for playback. The audio device on which to playback the output may be specified by name via the AudioPlayerConfiguration.DeviceName configuration parameter. The GetAvailableDevices static method may be used to enumerate the names of audio output devices currently available on the system.
Please note: This component uses Audio APIs that are available on Windows only.
