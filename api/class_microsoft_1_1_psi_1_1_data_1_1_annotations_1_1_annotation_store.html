Creates a new Annotation store and returns an AnnotationExporter instance which can be used to write streams to this store.

Parameters
pipelineThe Pipeline that owns the AnnotationExporter
nameThe name of the store to create
pathThe path to use. If null, an in-memory store is created.
definitionThe annotated event definition used to create and validate annotated events for this store.
createSubdirectoryIndicates whether to create a numbered subdirectory for each execution of the pipeline.
Returns
An AnnotationExporter instance that can be used to write streams.
