Detailed Description

Reads messages from a multi-stream store, either at the full speed allowed by available resources or at the desired rate specified by the Pipeline. Instances of this class can be created using the Store.Open method. The store metadata is available immediately after open (before the pipeline is running) via the AvailableStreams property.

◆ OpenAs< T >()

T Microsoft.Psi.Data.Importer.OpenAs< T > ( )

Opens a store by creating a dummy object of type T that exposes emitters that match to each stream in the store. This is useful for reading back stores that were previously created via Exporter.Write(T sensor, WhichStreams streamsToEmit, string[] streamNames) which is useful for saving streams from a sensor and then calling this method to play them back as if they were an attached device.
